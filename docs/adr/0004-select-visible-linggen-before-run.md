# Select a visible Linggen before the Run begins

Starting a Run presents three distinct Cultivator Candidates with visible roots and qualitative Affinity Grades; choosing one fixes the initial Linggen while exact affinities remain hidden. The seeded offering guarantees at least one single-root and one dual-root Candidate, with the third drawn from remaining profiles so restarting cannot reroll the tradeoff. Candidate names and portraits may vary, but Linggen is initially their only gameplay difference. We reject a hidden run-start roll revealed by Lingcao so players can make an informed roguelite opening decision; Lingcao now awakens the selected Linggen and grants the first Gongfa choice.

A Linggen profile is Candidate-eligible only when it supports at least three compatible Playable Gongfa. Stat-only or incomplete Gongfa configurations do not count and can never enter a Run offering.
